# Fotogram

Eine Foto-Galerie Website als Schulprojekt der Developer Akademie.

## Beschreibung
Fotogram ist eine persönliche Foto-Galerie, die Bilder in einer Thumbnail-Ansicht zeigt. Per Klick öffnet sich eine Großansicht mit Navigation.

## Features
- 12 Naturfotos dynamisch via JavaScript gerendert
- Lightbox mit Prev/Next Navigation
- Schließen per X, Hintergrund-Klick oder Esc-Taste
- Responsive Design

## Technologien
- HTML
- CSS
- JavaScript
